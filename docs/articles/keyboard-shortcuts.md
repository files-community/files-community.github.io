
# Keyboard Shortcuts

Files comes with built in keyboard shortcuts to make naviagting the app easier.

| Shortcut | Action |
| :---: | :---: |
| `Ctrl` + `C` | Copy |
| `Ctrl` + `X` | Cut |
| `Ctrl` + `V` | Paste |
| `Ctrl` + `T` | Create new tab |
| `Ctrl` + `W` | Close tab |
| `Ctrl` + `Tab` | Switch tab |
| `Ctrl` + `N` | New window |
| `Ctrl` + `Shift` + `N` | New file |
